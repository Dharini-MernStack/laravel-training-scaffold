# Day 01 – Laravel Setup

## Tasks Completed

- Installed PHP, Composer and MySQL in my laptop.
- Forked the given scaffold repo to my GitHub.
- Cloned my fork and ran `composer install`.
- Created `.env` file from `.env.example` and updated configs.
- Generated app key using `php artisan key:generate`.
- Created a local MySQL database and connected it in `.env`.
- Ran `php artisan migrate` successfully.
- Started the server using `php artisan serve` and checked the app in browser.
- Pushed my work to my forked repo.
